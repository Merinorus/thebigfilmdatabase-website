The Same Big Film Database - Work in Progress

IMPORTANT! This is a rewrite of this great website: The Big Film Database, which is still available (as of July, 2024).
This is a work in progress. The evolution is under discussion on the 35mm-compact.com forum (in French).
You can try the API, in alpha stage.
For the past few years, the database has only been updated sporadically. To prevent this work from disappearing, the original author has decided to place it under a Creative Commons BY-SA license on GitHub (with the exception of the images, which remain the exclusive property of their creators, used under "fair use").
