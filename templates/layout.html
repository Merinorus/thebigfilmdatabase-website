The (Modified) Big Film Database

IMPORTANT! I am not the original author of the Big Film Database.
This website is a modified version of The Big Film Database, which is still available (as of June, 2025) and whose the source code has recently been published on GitHub. Likewise, you can find on GitHub the source code of this modified website, as well as the modified database.

I'm trying to add features (search by DXN, show film edge barcode) and to improve search performances. You can try the API, in alpha stage.
The future of this website is under discussion on the 35mm-compact.com forum (in French).
